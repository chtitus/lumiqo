Parametres de l'application

Utilisee pour la generation de rapport IA. Les donnees identifiantes sont anonymisees avant envoi.
La reference sera generee : PREFIXE-ANNEE-NNN (ex: AUD-2025-003)
Un avertissement s'affichera 60 secondes avant le verrouillage.
Les données de test créent un client fictif "Clinique Saint-Luc" avec des écarts représentatifs.

Verification avant envoi a l'IA

Mode confidentiel actif (client HDS) - La generation IA est desactivee pour ce client. Utilisez uniquement les exports locaux (Word, texte, CSV).

Les donnees suivantes seront anonymisees automatiquement avant envoi :

  • ✅
    Nom client remplace par [CLIENT]
  • ✅
    Auditeur remplace par [AUDITEUR]
  • ✅
    Reference audit remplacee par [REF-AUDIT]
  • ✅
    Perimetre remplace par [PERIMETRE]
Votre responsabilite : Les champs libres (constats, preuves, ecarts, causes) sont envoyes tels quels. Verifiez qu'ils ne contiennent pas de noms de personnes, de systemes internes, ou toute donnee identifiante.
